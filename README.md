# 🎬 CineMatch — Movie Recommendation System

A content-based movie recommendation app built using the TMDB 5000 dataset.
Recommends similar movies based on shared cast & director.

## 🚀 Live Demo
[Try it here on Hugging Face Spaces](#)

## 🛠️ Tech Stack
- Python · Pandas · Scikit-learn
- Cosine Similarity (content-based filtering)
- Streamlit

## 📦 Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📂 Dataset
TMDB 5000 Credits — [Kaggle](https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata)
